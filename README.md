## 🚀 GitHub Pages

Strona jest hostowana za darmo na **GitHub Pages**.

### Jak to działa?

1. Każda zmiana wrzucona (`push`) do brancha `main` automatycznie aktualizuje stronę
2. Strona jest dostępna pod adresem `tmekgrabowski.pl`

### Jak edytować stronę?

Możesz edytować pliki bezpośrednio na GitHubie (kliknij plik → ikona ołówka ✏️),
albo pobrać projekt na komputer i edytować lokalnie.
